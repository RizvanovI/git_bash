1. Cоздайте файл bash2.txt: touch bash2.txt

2. Зайти в домашнюю директорию через терминал: cd

3. Создать папку test 3:  mkdir test3

4. Добавить в папку test 3 три файла 4, 5 и 6, в каждом из которых должно быть по 4 строки row1:
	echo -e "row1\nrow2\nrow3\nrow4" > test3/4.txt
	echo -e "row1\nrow2\nrow3\nrow4" > test3/5.txt
	echo -e "row1\nrow2\nrow3\nrow4" > test3/5.txt

5. Найдите строку row2 в файле 5: grep "row2" test3/5.txt

6. Найдите строку row в папке test3: grep -r "row" test3

7. Посчитайте сколько строк с содержимым row в файле 6: grep -c "row" test3/6.txt

8. Найдите файл 5 внутри папки test3: find test3 -name "5.txt"

9. Используя команду find, удалите файл 5: find test3 -name "5.txt" -delete

10. Используя команду echo, добавьте слово test в файл 4: echo "test" >> test3/4.txt

11. Замените слово test в файле 4 на fail: sed -i '' 's/fail/notfail/g' 4.txt

12. Добавьте в файл 4 слово test так, чтобы сохранилось содержимое: echo "test" >> test3/4.txt

13. Просмотрите все процессы для юзеров не только в консоли, которые происходят в системе: ps aux

14. Убейте процесс 666 в консоли: kill 666

15. Узнайте доступность ресурса rusau.net, используя ping: ping rusau.net

16. Отправьте 5 пакетов на сайт rusau.net: ping -c 5 rusau.net 

17. Используя GET и команду curl, получите информацию о зарегистрированных питомцах с любым статусом на https://petstore.swagger.io/: curl -X GET "https://petstore.swagger.io/v2/pet/findByStatus?status=sold"
