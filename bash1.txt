1. Cоздайте файл bash1.txt: touch bash1.txt

2. Открыть домашнюю директорию через терминал: cd

3. Определить имя папки, в которой вы находитесь: pwd

4. Создать внутри этой папки каталог с именем test1: mkdir test1

5. Перейти в папку test1: cd test1

6. Создать файл 1,2 и 3 внутри каталога test1: touch 1.txt 2.txt 3.txt

7. Проверить содержимое каталога test1: ls

8. Перейти в домашнюю директорию: cd

9. Создать папку test2 внутри домашней директории: mkdir test2

10. Удалить папку test2: rmdir test2

11. Удалить файл 2 из папки test1: cd test1
				   rm 2.txt
				   ls
12. Создать папку в домашней директории test3 и добавить в нее два файла: cd
								          mkdir test3
								          cd test3
								          touch 1.txt 2.txt
13. Удалить папку test3: cd
                         rm -rf test3

14. Создать папку test4 в домашней директории: mkdir test4

15. Переместить файлы 1 и 3 из папки test1 в папку test4: mv test1/1.txt test1/3.txt test4

16. Добавить в файл 1 три строки со словами line: echo "line" >> test4/1.txt
                                                  echo "line" >> test4/1.txt
                                                  echo "line" >> test4/1.txt

17. Посмотреть содержимое файла 1: cat test4/1.txt

18. Добавьте в файл 3 три строки со словами line: echo "line" >> test4/3.txt
                                                  echo "line" >> test4/3.txt
                                                  echo "line" >> test4/3.txt

19. Просмотрите содержимое двух файлов (1 и 3) сразу: cat test4/1.txt test4/3.txt

20. Используя один из редакторов замените все строки в файле 1: nano test4/1.txt
